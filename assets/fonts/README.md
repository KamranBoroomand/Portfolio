# Local Font Inventory

This directory contains self-hosted font files used by `assets/css/fonts.css`. Do not add private purchase receipts, license keys, order numbers, or provider account details to this public repository. If a commercial license exists, keep proof outside the repo and record only the non-sensitive outcome here.

## Manual Follow-Up

- Verify each non-JetBrains font against its original download package or foundry/license portal.
- Confirm the license explicitly allows public repository redistribution and `@font-face` web embedding.
- Keep private purchase/license documents outside this public repo.
- If a font cannot be verified, replace it with an open font that includes license text in the repo.
- `workbench-bled-scan.ttf` was removed because it was not referenced by `assets/css/fonts.css`
  and its exact redistribution source was not verified.

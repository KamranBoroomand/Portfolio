# Security Policy

## Reporting A Vulnerability

Please report security issues by email: `security@kamranboroomand.ir`.

Include the affected URL or file, a concise reproduction path, expected impact, and any safe proof of concept. Do not include secrets, destructive payloads, or data from third parties.

## Scope

This repository serves the static portfolio at `kamranboroomand.ir` and its local static assets. Runtime DNS, registrar, CDN, email routing, and hosting dashboard settings may be outside this repository unless a matching config file is present here.

## Handling

Reports are reviewed manually. Fixes will be prioritized by practical impact and verified with the repository quality gates before release.

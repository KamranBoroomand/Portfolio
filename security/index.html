Security Policy

Security Reporting

Use this page to report vulnerabilities affecting kamranboroomand.ir or the static assets served by this portfolio repository.

Contact

Email security reports to security@kamranboroomand.ir.

Include the affected URL or file, a concise reproduction path, expected impact, and any safe proof of concept.

Scope

In scope: this static portfolio, public assets, metadata files, and repository-managed deployment artifacts.

DNS, registrar, CDN, and mailbox routing are operational settings and may require manual verification outside this repository.

Guidelines

Please avoid destructive testing, persistence, social engineering, spam, or accessing data that does not belong to you. Reports are reviewed manually and fixes are prioritized by practical impact.

Privacy Analytics

This site uses no third-party analytics, ad-network pixels, session replay, fingerprinting, retargeting, or external tracking domains.

The optional first-party analytics request is same-origin, honors Do Not Track and Global Privacy Control, omits cookies, strips arbitrary query strings, and keeps only sanitized UTM campaign fields.

Visitors can opt out from the portfolio footer. The control stores localStorage.kb_analytics_opt_out = "1" in the browser.
